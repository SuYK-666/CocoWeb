# 猫狗图像分类项目 (基于 CNN 与 Grad-CAM 可视化)

## 项目简介

本项目是一个基于卷积神经网络 (CNN) 的深度学习项目，旨在实现对猫和狗图像的二分类任务。除了基础的训练与评估外，项目还引入了数据增强技术以抑制过拟合，并集成了 **Grad-CAM** 热力图技术，用于可视化模型在决策时的关注区域，增加了模型的可解释性。

## 项目结构

根据代码及目录结构，项目主要组成如下：

* **CNN_ImageClassification.py**: 核心 Python 脚本，包含数据预处理、模型构建、训练、评估及可视化代码。
* **数据结果/**: 存放模型生成的图表与可视化结果。
* `loss_acc_curve.png`: 训练过程中的准确率与损失曲线。
* `confusion_matrix.png`: 模型在验证集上的混淆矩阵。
* `gradcam_3x3_gallery.png` / `feature_map.png`: Grad-CAM 可视化图与特征图输出。
* **README.md**: 项目说明文档（本文件）。

## 环境配置

运行本项目需要以下 Python 环境及第三方库：

* **Python**: 3.x
* **深度学习框架**: TensorFlow / Keras
* **图像处理**: OpenCV (cv2), Pillow
* **数据分析与绘图**: NumPy, Matplotlib, Seaborn
* **机器学习工具**: Scikit-learn (用于生成混淆矩阵)

## 数据准备

项目使用经典的猫狗分类数据集。

1. **数据路径**: 代码中默认路径为 `D:\Downloads\EZ猫狗分类\cats_and_dogs_filtered`。运行前请根据实际存储位置修改 `base_path`。
2. **数据增强**: 为了提升泛化能力，对训练集应用了以下增强手段：
* 随机旋转 (25度)
* 水平翻转
* 宽度与高度平移 (15%)
* 像素值归一化 (Rescale 1/255)



## 模型架构

该模型采用了经典的 `Sequential` 序列结构，具体参数如下：

| 层级类型 | 参数/说明 |
| --- | --- |
| **卷积层 1** | 32个滤波器, (3,3) 卷积核, ReLU 激活 |
| **池化层 1** | 2x2 最大池化 |
| **卷积层 2** | 64个滤波器, (3,3) 卷积核, ReLU 激活 |
| **池化层 2** | 2x2 最大池化 |
| **卷积层 3** | 128个滤波器, (3,3) 卷积核, ReLU 激活 |
| **池化层 3** | 2x2 最大池化 |
| **卷积层 4** | 128个滤波器, (3,3) 卷积核, ReLU 激活 |
| **池化层 4** | 2x2 最大池化 |
| **展平层** | 将多维特征转为一维向量 |
| **Dropout层** | 丢弃率为 0.5 (防止过拟合) |
| **全连接层** | 512个神经元, ReLU 激活 |
| **输出层** | 1个神经元, Sigmoid 激活 (输出为狗的概率) |

## 训练参数

* **优化器**: Adam (学习率: 1e-4)
* **损失函数**: Binary Crossentropy (二元交叉熵)
* **训练轮数 (Epochs)**: 10
* **批大小 (Batch Size)**: 32

## 结果分析与可视化

训练完成后，项目会自动生成以下反馈：

1. **性能曲线**: 绘制训练集与验证集的 Accuracy (准确率) 和 Loss (损失) 变化图。
2. **混淆矩阵**: 通过 Seaborn 绘制热力图，直观展示猫、狗分类的分类精度与误判情况。
3. **Grad-CAM 可视化**: 利用梯度信息生成热力图，展示模型在识别猫或狗时，主要参考了图像的哪些局部特征。

## 如何运行

1. 确保已安装所有依赖库。
2. 下载猫狗分类数据集，并更新 `CNN_ImageClassification.py` 中的 `base_path` 路径。
3. 在终端或 IDE 中运行脚本：
```bash
python CNN_ImageClassification.py

```
4. 训练结束后，模型将自动保存为 `cat_dog_model.h5`。
